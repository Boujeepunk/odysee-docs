# Tipps zum Empfang von Bargeld

## Ihr Bankkonto einbinden

Um ein Bargeldtipp zu erhalten, müssen Sie ein Bankkonto hinzufügen. Eine Aufschlüsselung der Gebühren für Bargeldtransaktionen finden Sie in unserem [Zahlungsplan](https://help.odysee.tv/de/payments-chart/)

Um ein Bankkonto hinzuzufügen, müssen Sie zunächst Folgendes eröffnen <OdyseeButton url="https://odysee.com/$/wallet"><OdyseeIcon type="credits" /> Your Wallet</OdyseeButton>. Ihre Brieftasche befindet sich oben rechts auf dem Bildschirm in der Menüleiste.

![](/img/en/category-basics17.png)

In der mobilen Version finden Sie es am unteren Ende des Seitenleistenmenüs.

![](/img/en/category-basics18.png)

Von hier aus wählen Sie <OdyseeButton url="https://odysee.com/$/settings/tip_account"><OdyseeIcon type="cogwheel" /> Bankkonten</OdyseeButton>.

![](/img/en/category-basics19.png)

Das führt Sie hierher, wo Sie Folgendes auswählen <PrimaryOdyseeButton url="https://odysee.com/$/settings/tip_account"><OdyseeIcon type="dollar" /> Verbinden Sie Ihr Bankkonto</PrimaryOdyseeButton>.

![](/img/en/category-basics20.png)

Dies führt Sie außerhalb von Odysee nach Stripe.

![](/img/en/category-basics21.png)

Von hier aus geben Sie einfach die gewünschten Informationen ein.

## Entgegennahme von Trinkgeldern

Nachdem Ihr Bankkonto hinzugefügt wurde und Sie Trinkgelder erhalten haben, können Sie den Betrag, den Sie erhalten haben, in Ihrer Brieftasche verfolgen.

Alle 2-3 Werktage werden die an Ihre Odysee-Brieftasche gesendeten Trinkgelder automatisch ausgezahlt und auf Ihr Bankkonto überwiesen.

:::info Information

Damit die Anbindung Ihres Bankkontos an Odysee erfolgreich ist, benötigen Sie ein Dollar-Bankkonto. In einigen Regionen oder Ländern ist dies noch nicht möglich, so dass Sie die Verfügbarkeit in Ihrem Land prüfen müssen, wenn Sie Ihr Bankkonto oder Ihre Karte anschließen.

:::
