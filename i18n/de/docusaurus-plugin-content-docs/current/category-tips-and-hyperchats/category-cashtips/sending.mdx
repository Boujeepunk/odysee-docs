# Tipps zum Senden von Bargeld

## Hinzufügen einer Zahlungsmethode

Bevor Sie ein Bargeldtipp senden können, müssen Sie eine Zahlungsmethode hinzufügen. Eine Übersicht über die Gebühren für Bargeldtransaktionen finden Sie in unserem [Zahlungsplan](https://help.odysee.tv/de/payments-chart/)

Um eine Karte hinzuzufügen, müssen Sie zunächst Ihre Brieftasche öffnen. Die Brieftasche befindet sich oben rechts auf dem Bildschirm in der Menüleiste.

![](/img/en/category-basics17.png)

In der mobilen Version finden Sie es am unteren Ende des Seitenleistenmenüs.

![](/img/en/category-basics18.png)


Hier wählen Sie **Zahlungsmethoden** aus.

![](/img/en/category-basics23.png)

Auf dem Bildschirm "Zahlungsmethoden" geben Sie einfach Ihren vollständigen Namen, die Kartennummer, das Ablaufdatum und den dreistelligen CVC-Code ein.

![](/img/en/category-basics24.png)

Wenn Sie bereits eine Karte hinzugefügt haben, wird diese auf dem Bildschirm Zahlungsmethoden angezeigt.

## Bargeldtipps senden

Um ein Trinkgeld zu senden, klicken Sie auf die Schaltfläche **Support** unter einem Upload.

![](/img/en/category-monetization6.png)

Diese Option ist auch auf den Kanälen verfügbar.

![](/img/en/category-monetization7.png)

Wählen Sie in diesem Menü die Schaltfläche **Tipp** mit dem USD-Symbol, wählen Sie einen Betrag und klicken Sie auf **Tipp senden**.

:::info Informationen

Damit die Anbindung Ihres Bankkontos an Odysee erfolgreich ist, benötigen Sie ein Dollar-Bankkonto. In einigen Regionen oder Ländern ist dies noch nicht möglich, so dass Sie die Verfügbarkeit in Ihrem Land prüfen müssen, wenn Sie Ihr Bankkonto oder Ihre Karte anschließen.

:::
