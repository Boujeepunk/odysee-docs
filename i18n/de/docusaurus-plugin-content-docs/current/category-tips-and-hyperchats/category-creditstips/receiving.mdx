# Empfang von Credits-Tipps

Wenn ein Credits-Tipp beim Betrachten eines Inhalts gesendet wird, erhöht er den Credits-Einsatz dieses Inhalts, bis er freigeschaltet wird. Solange er nicht freigeschaltet ist, kann er nicht an anderer Stelle auf Odysee verwendet werden.

Um einen Tipp freizuschalten, gehen Sie zunächst zur Brieftasche, indem Sie auf **Ihre Brieftasche** klicken.

![](/img/en/category-wallet1.png)

Auf diesem Bildschirm wählen Sie **Transaktionen** und ändern den **Typ** in Empfangen.

![](/img/en/odysee-monetization9.png)

Wenn Sie den Boost sehen, den Sie erstatten möchten, klicken Sie einfach auf das Symbol zum Entsperren neben "Tipp", um den Tipp zu entsperren.

**Bitte beachten Sie: Sobald ein Tipp freigeschaltet ist, wird er den Inhalt, für den er ursprünglich getippt wurde, nicht mehr verstärken.

Weitere Informationen über Content-Boosts und das Odysee-Einsatzsystem finden Sie in unserem Abschnitt über Einsätze, indem Sie [hier klicken](https://help.odysee.tv/de/category-blockchain/category-staking/naming).
