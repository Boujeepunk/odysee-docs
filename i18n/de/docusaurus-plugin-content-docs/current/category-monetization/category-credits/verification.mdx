# Verifizierung

Eine Verifizierung ist für die Nutzung von Odysee nicht erforderlich, wird aber dringend empfohlen. Die Verifizierung ermöglicht es Ihnen, einige Credits zu erhalten, indem Sie Odysee einfach benutzen. Wir verlangen eine Verifizierung, um einige Credits zu erhalten, um Missbrauch zu vermeiden.

Die meisten Konten werden automatisch verifiziert. Abhängig von Faktoren wie Standort, VPN-Nutzung usw. werden Sie möglicherweise um zusätzliche Informationen zur Verifizierung gebeten. Hierfür haben Sie mehrere Möglichkeiten:

<Image img="/img/en/category-basics6.png"/>

**Über die Telefonnummer**

Ihre erste Möglichkeit ist die Verifizierung über Ihre Mobiltelefonnummer. Wir senden Ihnen eine Textnachricht mit einem Code. Nicht alle Regionen sind für die Verifizierung per Telefonnummer geeignet.

**Mit Karte** 

Sie können sich auch mit einer Kredit- oder Debitkarte verifizieren, die mit einem Dollar-Bankkonto verbunden ist. Odysee speichert weder Ihre Kartendaten noch werden Ihnen Gebühren berechnet. Es kann sein, dass auf Ihrem Bankkonto eine Zahlungsermächtigung in Höhe von 1 Dollar erscheint, die jedoch wieder auf Ihr Konto zurückgebucht wird.

**Mit E-Mail**

Sie können eine Verifizierung Ihres Odysee-Kontos beantragen, indem Sie eine E-Mail an hello@odysee.com senden.

Zurzeit werden Verifizierungsanfragen manuell geprüft, was bedeutet, dass es einige Stunden bis mehrere Tage dauern kann, bis sie genehmigt werden.
