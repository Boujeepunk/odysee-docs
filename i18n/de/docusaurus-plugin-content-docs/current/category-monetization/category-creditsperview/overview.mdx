# Übersicht

Durch den Erhalt von Credits pro View auf Odysee erhalten Ersteller Credits für jeden verifizierten View. Eine verifizierte Ansicht ist eine Ansicht von einem verifizierten Nutzer. Das bedeutet, dass wir überprüft haben, dass Sie eine reale Person sind und somit Missbrauch verhindern. Um Missbrauch zu verhindern, werden nur die ersten 7 Views eines verifizierten Nutzers pro Tag monetarisiert.

## Wie viel bekomme ich pro Ansicht?

Um Ihre Credits pro Ansicht zu überprüfen, navigieren Sie zur Seite mit den Kanälen, indem Sie auf das Profilbildsymbol klicken und im Dropdown-Menü **Kanäle** auswählen.

![](/img/en/category-channels1.png)

Ihre Credits pro Ansicht werden rechts neben Ihrem Kanal angezeigt.

![](/img/en/category-monetization4.png)

**Hinweis: Ihre Rate kann sich ändern und basiert auf einer Reihe von Variablen, wie z. B. der durchschnittlichen Sehdauer, der durchschnittlichen Anzahl der Aufrufe, der Art des Inhalts, dem Engagement und dem Standort des Erstellers**

## Tracking-Gutschriften

Klicken Sie auf das Profilbild-Symbol in der oberen rechten Ecke und rufen Sie die Seite auf.

![](/img/en/category-monetization1.png)

Auf dieser Seite wird die Credits-Ansicht als *Inhaltsansicht Tipps* angezeigt.

![](/img/en/category-monetization3.png)
