# Berechtigung

Die Berechtigung, Credits pro Ansicht zu erhalten, hängt von den folgenden Bedingungen ab:

- Der Inhalt muss Ihr eigener Originalinhalt sein und darf nicht von den Originalwerken anderer abgeleitet sein.
- Der Inhalt und der Kanal müssen korrekt getaggt sein.
- Das Konto muss mindestens eine Woche alt sein.

Die folgenden Arten von Inhalten sind nicht für den Erhalt von Credits pro Ansicht geeignet:

- Verstößt gegen unsere Community-Richtlinien.
- Urheberrechtlich geschützte Werke.
- Illegale Inhalte.
- Kontroverse Inhalte (echte Gewalt, Folter, Missbrauch, Aufstachelung zum Hass) - Inhalte, die eines oder mehrere dieser Elemente enthalten, aber keine Nachrichten/Bildungsinhalte sind.
- Spam.
- Reuploads oder abgeleitete Werke auf andere öffentliche/urheberrechtsfreie/kreative Commons-Inhalte.
- Bezogen auf Airdrops/Gratisgeld.

Wenn Sie der Meinung sind, dass Ihr Kanal nicht monetarisiert wurde, obwohl dies der Fall sein sollte, senden Sie uns bitte eine [E-Mail] (hello@odysee.com) mit einem Link und einer Beschreibung Ihres Kanals.
