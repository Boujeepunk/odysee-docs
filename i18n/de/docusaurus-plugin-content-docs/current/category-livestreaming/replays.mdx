# Wiederholungen hochladen

Derzeit gibt es keine Option zur automatischen Veröffentlichung von Livestream-Wiederholungen.

Um einen früheren Livestream zu veröffentlichen, der als Wiederholung in Ihrem Odysee-Konto gespeichert wurde, müssen Sie zur Seite "Go Live" in Ihrem Odysee-Konto zurückkehren und eine der Optionen auswählen, bei der Sie die Wiederholung auf Ihrem Kanal veröffentlichen möchten.

![](/img/en/category-livestreaming11.png)

Von hier aus wählen Sie **Wiedergabe auswählen**. Wenn keine Wiederholungen angezeigt werden, können Sie eine Aktualisierung durchführen, indem Sie **Nach Wiederholungen suchen** auswählen.

Auf der Registerkarte "Wiederholung auswählen" wählen Sie die Wiederholung des vorherigen Livestreams aus, die Sie veröffentlichen möchten, geben die angeforderten Informationen in das Formular ein und veröffentlichen dann die Wiederholung, indem Sie unten auf der Seite auf die Schaltfläche "Erstellen" klicken.

# Beitragswiederholung vom Livestream-Beitrag

Möchten Sie eine Wiederholung eines früheren Livestreams mit derselben URL veröffentlichen, die Sie für den Livestream-Beitrag verwendet haben? Schau mal.

:::info Notiz:

Bei dieser Option können Sie einen Replay-Beitrag nur einmal mit derselben URL veröffentlichen. Dies liegt daran, dass es sich um denselben Livestream-Beitrag handelt, der zuvor veröffentlicht wurde. Sie können den Beitrag bearbeiten und sogar das Video ersetzen, aber Sie können die URL nicht ändern.

:::

Ein Livestream-Beitrag verfügt über eine von Ihnen erstellte benutzerdefinierte URL, die Sie zum Posten einer Wiederholung verwenden können. Der ursprüngliche Beitrag, den Sie im Livestream verwendet haben, speichert auch den Chat, der dann als Kommentare zum Beitrag veröffentlicht wird. Wenn Sie sich jedoch dafür entscheiden, unabhängig eine Wiederholung zu posten (ein anderer Beitrag/eine andere URL), wird der Chat nicht als Kommentar angezeigt.

Um eine Wiederholung mit derselben URL zu veröffentlichen, müssen Sie auf der Seite "Live gehen" zur Registerkarte "Lokale Einrichtung" gehen und die folgenden Schritte ausführen:

![](https://thumbs.odycdn.com/b17f6832fb4e77e63718a649cccf5f17.webp)

1. Klicken Sie auf den Livestream-Beitrag.
2. Klicken Sie auf die Schaltfläche "Wiedergabe aktualisieren oder veröffentlichen".
3. Wählen Sie im Formular die Option "Wiedergabe verwenden".
4. Optional: Sie können eine Wiederholung manuell von Ihrem Gerät hochladen.
5. Wenn im Formular alles korrekt ist, klicken Sie auf die Schaltfläche "Speichern".

![](https://thumbs.odycdn.com/29de2849511dae624eae40a2e2bc426f.webp)

![](https://thumbs.odycdn.com/f2a229180dd741f335537d8e5a82cffa.webp)

Nach einigen Minuten wird der Beitrag mit der Wiederholung auf Ihrem Kanal veröffentlicht.

:::info Notiz:

In einigen Fällen können Wiederholungen zu Aufnahme- oder Verarbeitungsfehlern führen. Sie können uns eine E-Mail an hello@odysee.com senden, damit wir Ihnen helfen können. Darüber hinaus ist es nicht möglich, mehrere Wiederholungen für einen einzelnen Beitrag zusammenzuführen. Sie müssen jede Wiederholung separat veröffentlichen.

:::
