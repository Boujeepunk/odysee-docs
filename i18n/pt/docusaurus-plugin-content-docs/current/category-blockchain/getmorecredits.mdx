# Como faço para receber alguns mais Créditos?

## Convites

Você pode receber alguns Créditos convidando outras pessoas para o Odysee. Em odysee.com/$/invite, você receberá um link de convite.

Compartilhe este link e, para cada usuário único que clicar nele, você receberá um pouco de Créditos.

Isso é especialmente útil para criadores que estão compartilhando seu canal Odysee com seus espectadores. Em vez de fornecer o link básico do seu canal, forneça o link de convite.

## Dicas em Créditos

As dicas em Créditos dos espectadores também são uma maneira de receber mais Créditos. Quando as dicas são enviadas pela primeira vez, elas aumentam o conteúdo sobre o qual foram informadas e não estarão disponíveis para uso até que sejam desbloqueadas. Para desbloquear uma dica, primeiro acesse a carteira clicando em **Sua Carteira**.

Nessa tela, você selecionará **Transações** e alterará o **Tipo** para Recebido.

Quando você vir o aumento que deseja reembolsar, basta clicar no ícone de desbloqueio ao lado de "Dica" e você desbloqueará a dica.

**Observação: uma vez que uma dica é desbloqueada, ela não irá mais impulsionar o conteúdo originalmente informado.**
