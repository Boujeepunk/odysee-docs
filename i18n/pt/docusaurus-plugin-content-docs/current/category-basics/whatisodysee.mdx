# O que é a Odysee?

A Odysee é uma plataforma de mídia baseada em blockchain. Hospedamos todos os tipos de mídia, como imagens, artigos, PDFs, arquivos de áudio, etc., mas somos mais conhecidos por hospedar vídeos.

A Odysee busca recapturar o espírito da Internet do início dos anos 2000. Em vez de favorecer o conteúdo corporativo, como talk shows noturnos, redes de televisão e noticiários de TV, o Odysee é um lugar para todos, inclusive para criadores independentes.

Referir-se ao Odysee como “baseado em blockchain” não é apenas um slogan. Todo o conteúdo da Odysee é hospedado na tecnologia blockchain, da qual a Odysee faz uso. Para os interessados em blockchain, isso é muito legal. Para aqueles que não sabem nada sobre blockchain, não há problema algum, pois a Odysee não requer nenhum conhecimento sobre blockchain para ser usada.
