# Envie dicas em Créditos

Para enviar uma dica, clique no botão **Suporte** abaixo de um upload.

![](/img/en/category-monetization6.png)

Esta opção também está disponível em canais.

![](/img/en/category-monetization8.png)

Nesse menu, você selecionará o botão **Dica** com o ícone Créditos, selecionará um valor e clicará em **Enviar dica**.

Essa dica de Créditos aumentará o conteúdo/canal que está sendo sugerido, até que o criador decida desbloqueá-lo para uso em outro lugar no Odysee.
