# Verificação

A verificação não é necessária para usar o Odysee, mas é altamente recomendada. A verificação permite que você receba alguns Créditos simplesmente usando o Odysee. Exigimos verificação para receber alguns Créditos, para evitar abusos.

A maioria das contas é verificada automaticamente. Dependendo de fatores como localização, uso de VPN, etc., podem ser solicitadas informações adicionais para verificação. Para fazer isso, você tem algumas opções:

<Image img="/img/en/category-basics6.png"/>

**Através do número de telefone**

Sua primeira opção é verificar através do seu número de celular. Enviaremos uma mensagem de texto com um código. Nem todas as regiões são elegíveis para verificação de número de telefone.

**Via Cartão** 

Você também pode verificar usando um cartão de crédito ou débito conectado a uma conta bancária em dólares. A Odysee não armazena as informações do seu cartão nem cobraremos nenhuma taxa. Você poderá ver uma autorização de pagamento de US$ 1 em sua conta bancária, mas ela será devolvida à sua conta.

**Via email**

Você pode solicitar a verificação da sua conta Odysee enviando um email para hello@odysee.com.

Neste momento, as solicitações de verificação são revisadas manualmente, o que significa que podem levar de algumas horas a vários dias para serem aprovadas.
