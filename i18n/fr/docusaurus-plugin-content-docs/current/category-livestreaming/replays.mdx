# Téléchargement de rediffusions

Pour le moment, il n'existe aucune option permettant de publier automatiquement les rediffusions du livestream.

Pour publier un livestream précédent qui a été enregistré en tant que rediffusion dans votre compte Odysee, vous devez revenir à la page "Go Live" de votre compte Odysee pour sélectionner l'une des options dans lesquelles vous souhaitez publier la rediffusion sur votre chaîne.

![](/img/en/category-livestreaming11.png)

À partir de là, vous sélectionnerez **Choisir Replay**. Si aucune rediffusion n'apparaît, vous pouvez actualiser en sélectionnant **Vérifier les rediffusions**.

Dans l'onglet "Choisir le replay", vous sélectionnerez le replay du livestream précédent que vous souhaitez publier, saisirez les informations demandées dans le formulaire, puis publierez le replay en cliquant sur le bouton "Créer" en bas de la page.

# Télécharger la rediffusion d'une publication

Souhaitez-vous publier une rediffusion d'un livestream précédent avec la même URL que celle que vous avez utilisée pour la publication du livestream? Regarde.

:::info Note:

Pour cette option, vous ne pouvez publier qu’une seule fois une publication en rediffusion avec la même URL. En effet, il s’agit du même message en direct publié précédemment. Vous pouvez modifier la publication et même remplacer la vidéo, mais vous ne pouvez pas modifier l'URL.

:::

Une publication en direct comporte une URL personnalisée créée par vous, que vous pouvez utiliser pour publier une rediffusion. La publication originale que vous avez utilisée dans le livestream enregistre également le chat qui est ensuite publié sous forme de commentaires sur la publication. Cependant, si vous choisissez de publier une rediffusion indépendamment (une autre publication/URL), le chat ne s'affichera pas sous forme de commentaires.

Pour publier un replay avec la même URL vous devez vous rendre dans l'onglet "Configuration locale" de la page "Go Live" et suivre ces étapes :

![](https://thumbs.odycdn.com/b17f6832fb4e77e63718a649cccf5f17.webp)

1. Cliquez sur la publication en direct.
2. Cliquez sur le bouton "Mettre à jour ou publier la relecture".
3. Dans le formulaire, sélectionnez l'option "Utiliser Replay".
4. Facultatif: vous pouvez télécharger une rediffusion manuellement depuis votre appareil.
5. Si tout est correct dans le formulaire, cliquez sur le bouton "Enregistrer".

![](https://thumbs.odycdn.com/29de2849511dae624eae40a2e2bc426f.webp)

![](https://thumbs.odycdn.com/f2a229180dd741f335537d8e5a82cffa.webp)

Après quelques minutes, le post avec le replay sera publié sur votre chaîne.

:::info Note:

Dans certains cas, les replays peuvent provoquer des erreurs d'enregistrement ou de traitement. Vous pouvez nous envoyer un e-mail à hello@odysee.com afin que nous puissions vous aider. De plus, il n’est pas possible de fusionner plusieurs Replays pour une seule publication. Vous devez publier chaque Replay séparément.

:::
