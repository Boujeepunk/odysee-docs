## Votre chaîne n'est pas candidate à la synchronisation

Si vous essayez de synchroniser votre chaîne YouTube avec Odysee et que l'erreur: **"Votre chaîne n'est pas candidate à la synchronisation"** s'affiche, il est possible que la chaîne ne remplisse pas certaines conditions minimales pour être synchronisée. Si la chaîne remplit les conditions requises mais ne peut toujours pas être synchronisée, veuillez nous envoyer un e-mail.

Consultez la page [Exigences et limites](https://help.odysee.tv/fr/category-syncprogram/) de la documentation.

## Chaîne connectée à un autre compte

Si vous essayez de synchroniser une chaîne YouTube avec Odysee et que le message **"La chaîne est connectée à un autre compte"** s'affiche, vous devrez peut-être évaluer certaines de ces options:

1. Le compte a été supprimé avec la chaîne YouTube connectée à ce compte, mais la chaîne n'a pas encore été synchronisée. Vous devrez envoyer un e-mail pour essayer de résoudre le problème.
2. La chaîne a été synchronisée avec un autre compte mais vous avez essayé de la synchroniser à nouveau. Désolé, il n'est pas possible de synchroniser la chaîne avec un autre compte si elle a déjà été synchronisée.
3. La chaîne est synchronisée avec un autre compte mais la chaîne a été supprimée. Dans ces cas, il est possible que la chaîne ne puisse pas être à nouveau synchronisée sur un autre compte.
4. Essayer de synchroniser une chaîne YouTube avec une chaîne créée manuellement dans Odysee. Désolé, le transfert de contenu se fera vers la chaîne créée automatiquement par synchronisation. Cependant, vous pouvez éditer une vidéo sur votre chaîne Odysee et l'enregistrer sur une autre chaîne.

**Ici, vous pouvez découvrir comment procéder :** [Sélection des chaînes](https://help.odysee.tv/fr/category-uploading/category-fileuploads/settings/)

:::info

Si vous pensez que votre cas est différent des erreurs de synchronisation mentionnées ici, vous pouvez nous envoyer un courriel à hello@odysee.com. Nous nous ferons un plaisir d'examiner votre cas.

:::

## Erreur lors du téléchargement du contenu

Dans certains cas, des messages d'erreur inattendus peuvent apparaître lors du téléchargement de contenu. Certaines de ces erreurs inattendues peuvent être corrigées en apportant quelques modifications à la publication.

1. **"Solde insuffisant"**. Le message suggère que vous n'avez pas suffisamment de Crédits pour publier ou booster du contenu. Veuillez vérifier que vous disposez de suffisamment de Crédits sur la page du portefeuille dans la section Solde des dépenses immédiates. Vérifiez que votre message ne dépasse pas le montant des Crédits que vous avez dans votre portefeuille. Si vous avez besoin de plus de Crédits pour des dépenses immédiates, vous pouvez modifier le contenu publié ou révoquer les Boosts des transactions sur la page du portefeuille.
2. **"La transaction a été rejetée par les règles du réseau. TX rejetée: la taille (nombre) du script dépasse la limite."** Vous avez peut-être dépassé le nombre de caractères dans la description de votre contenu. Vous devrez modifier le nombre de caractères et réessayer de télécharger le contenu.
3. **"Le média n'a pas pu être chargé, soit parce que le serveur ou le réseau est en panne, soit parce que le format n'est pas pris en charge."** Il est possible que vous ayez mis en ligne une vidéo avec un encodage HEVC (H.265). ou un autre codec non pris en charge. Nous n'avons actuellement pas de transcodage automatique lors du téléchargement, les vidéos sont donc publiées dans le format original. Nous vous recommandons d'utiliser l'encodage vidéo H264 car il est pris en charge par pratiquement tous les navigateurs et appareils.

**Vous pouvez trouver notre petit guide ici:** [Encodage de vidéos](https://help.odysee.tv/fr/category-uploading/category-fileuploads/encoding/)

4. **"Une erreur réseau s'est produite, mais la publication est peut-être terminée. Attendez quelques minutes, puis vérifiez votre page de téléchargements ou votre portefeuille pour confirmer."** Cette erreur inattendue peut être provoquée par une interruption de le contenu se charge par connexion, mais il peut avoir été publié avec succès. Vous pouvez le vérifier en visitant la page Téléchargements.

## Erreur de navigation, de vérification ou de captcha

Dans certains cas, les navigateurs Web dotés de boucliers publicitaires et d'autres scripts qui bloquent les fonctionnalités activés peuvent provoquer des erreurs de navigation et d'expérience dans Odysee.

Nous vous recommandons de désactiver temporairement les boucliers ou les scripts dans votre navigateur Web compatible Odysee pour profiter du contenu ou prendre d'autres mesures nécessaires. Cela aidera également à vérifier le compte avec le système captcha.

## Odysee vide dans le navigateur Web

Si vous ouvrez Odysee et voyez un écran vide, nous vous recommandons de vider le cache de votre navigateur Web et de réessayer. Dans certains cas, vous devrez attendre quelques minutes et tout redeviendra normal. En attendant, vous pouvez utiliser Odysee temporairement dans un autre navigateur Web.

## Erreur de lecture

Dans certains cas, une erreur peut s'afficher lors de la lecture vidéo, **"la lecture du média a été interrompue en raison d'un problème de corruption ou parce que les fonctionnalités du média utilisé ne sont pas prises en charge par votre navigateur"**. Cette erreur peut signifier que la vidéo a été téléchargée sur Odysee avec un codec non pris en charge ou une compression de codec incorrecte. Vous pouvez essayer de regarder la vidéo à partir d'un autre navigateur Web, mais la lecture ou le flux risque de ne pas être convivial.

Si la vidéo n'est pas optimisée pour le web, il est nécessaire de télécharger une grande partie de la vidéo avant de pouvoir la lire. Vous pouvez optimiser les vidéos sur le Web à l'aide de Handbrake. Vous pouvez en savoir plus ici : [Encodage des vidéos](https://help.odysee.tv/fr/category-uploading/category-fileuploads/encoding/).

**Pour le contenu vidéo, utilisez des MP4 au format H264/AAC et un débit binaire convivial (inférieur à 8 Mbps) pour un streaming plus fiable.**

## Portefeuille vide

Si vous accédez à la page de votre portefeuille et voyez un message de portefeuille vide, cela peut être dû aux raisons suivantes :

1. Vérifiez que votre compte a été vérifié pour recevoir des Crédits.
2. Accédez à la page Crédits et recevez des Crédits pour regarder du contenu ou certaines tâches supplémentaires.
3. Si vous avez des Crédits mis en jeu sur le contenu, vous pouvez débloquer des Crédits en modifiant la publication pour envoyer des Crédits comme dépense immédiate sur la page du portefeuille.
4. Vérifiez que vous n'avez pas d'autre contenu Boosting de Crédits. Vérifiez le filtre de transaction sur la page du portefeuille.
5. Peut-être vous êtes-vous connecté à un autre compte ? Vérifiez vos informations d'identification.
6. Dans certains cas, il est nécessaire de synchroniser le portefeuille. Contactez nous s'il vous plait.

:::info

Si vous ne trouvez pas d'erreur inattendue ici, il peut s'agir d'une interruption temporaire d'un service. Envoyez-nous un courriel à hello@odysee.com avec plus de détails sur l'erreur que vous rencontrez. Des captures d'écran peuvent également être utiles.

:::
