# Vue d'ensemble

Recevoir des Crédits par vue sur Odysee donne aux créateurs des Crédits pour chaque vue vérifiée. Une vue vérifiée est une vue d'un utilisateur vérifié. Cela signifie que nous avons vérifié que vous êtes une personne réelle et que nous évitons ainsi les abus. En plus de prévenir les abus, les créateurs ne monétiseront chaque jour que les 7 premières vues d'un utilisateur vérifié.

## Combien est-ce que je reçois par vue?

Pour vérifier vos Crédits par vue, accédez à la page des chaînes en cliquant sur l'icône de photo de profil et en sélectionnant **Chaînes** dans le menu déroulant.

![](/img/en/category-channels1.png)

Vos Crédits par vue s'afficheront à droite de votre chaîne.

![](/img/en/category-monetization4.png)

**Remarque: Votre tarif est susceptible de changer et est basé sur un certain nombre de variables, notamment la durée moyenne de visionnage, le nombre moyen de vues, le type de contenu, l'engagement et l'emplacement du créateur**.

## Crédits de suivi

Cliquez sur l'icône de photo de profil dans le coin supérieur droit et accédez à la page.

![](/img/en/category-monetization1.png)

À partir de cette page, la vue Crédits s'affichera sous la forme *Conseils pour afficher le contenu*.

![](/img/en/category-monetization3.png)
