# Envoi de pourboire en Crédits

Pour envoyer un pourboire, vous cliquerez sur le bouton **Support** sous une mise en ligne.

![](/img/en/category-monetization6.png)

Cette option est également disponible sur les chaînes.

![](/img/en/category-monetization8.png)

Dans ce menu, vous sélectionnerez le bouton **Pourboire** avec l'icône Crédits, sélectionnez un montant et cliquez sur **Envoyer le pourboire**.

Cette astuce de crédits augmentera le contenu/la chaîne faisant l'objet d'une astuce, jusqu'à ce que le créateur choisisse de le déverrouiller pour une utilisation ailleurs sur Odysee.
