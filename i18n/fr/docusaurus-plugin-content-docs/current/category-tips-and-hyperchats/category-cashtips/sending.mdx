# Envoyez des pourboires en espèces

## Adding a Payment Method

Avant d'envoyer un pourboire en espèces, vous devrez ajouter un mode de paiement. Pour connaître le détail des frais de transaction en espèces, consultez notre [tableau des paiements](https://help.odysee.tv/fr/payments-chart/)

Pour ajouter une carte, vous devrez d'abord ouvrir votre portefeuille. Votre portefeuille se trouve en haut à droite de l'écran, dans la barre de menu.

![](/img/en/category-basics17.png)

Sur la version mobile, vous le trouverez en bas du menu de la barre latérale.

![](/img/en/category-basics18.png)

À partir de là, vous sélectionnerez **Modes de paiement**.

![](/img/en/category-basics23.png)

Sur l'écran Modes de paiement, vous saisirez simplement votre nom complet, votre numéro de carte, sa date d'expiration et son code CVC à 3 chiffres.

![](/img/en/category-basics24.png)

Si vous avez déjà ajouté une carte, elle apparaîtra sur l'écran Modes de paiement.

## Conseils d’envoi d’argent

Pour envoyer un pourboire, vous cliquerez sur le bouton **Assistance** sous une mise en ligne.

![](/img/en/category-monetization6.png)

Cette option est également disponible sur les chaînes.

![](/img/en/category-monetization7.png)

Dans ce menu, vous sélectionnerez le bouton **Pourboire** avec le symbole USD, sélectionnez un montant et cliquez sur **Envoyer le pourboire**.

:::info Information

Pour que votre connexion à votre compte bancaire réussisse sur Odysee, vous devrez disposer d'un compte bancaire en dollars. Certaines régions ou pays ne sont pas encore éligibles, vous devrez donc vérifier la disponibilité dans votre pays lors de la connexion de votre compte bancaire ou de votre carte.

:::
