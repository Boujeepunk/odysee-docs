# Envío de propinas LBC

Para enviar una propina, deberá hacer clic en el botón **Soporte** debajo del contenido.

![](/img/en/category-monetization6.png)

Esta opción también está disponible en los canales.

![](/img/en/category-monetization8.png)

En este menú, seleccionará el botón **Propina** con el ícono LBC, seleccionará una cantidad y hará clic en **Enviar propina**.

Esta propina de LBC mejorará el contenido/canal que se está promocionando, hasta que el creador decida desbloquearlo para usarlo en otro lugar de Odysee.
